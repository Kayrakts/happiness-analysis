# Küresel Mutluluk Düzeyini Etkileyen Faktörlerin Analizi

## Proje Hakkında
World Happiness Report 2019 veri seti kullanılarak 156 ülkenin mutluluk 
skorlarını etkileyen faktörler incelenmiştir. Pearson korelasyonu, basit 
regresyon ve bağımsız örneklem t-testi uygulanmıştır.

## Kullanılan Araçlar
- JASP
- Tableau

## Veri Seti
[World Happiness Report 2019 - Kaggle](https://www.kaggle.com/datasets/unsdsn/world-happiness)
